## 🕹️ 游戏介绍

「荆棘收割者」是一款受《吸血鬼幸存者》启发的割草（Survivor-like）游戏。

- **玩法**：在无尽的敌人浪潮中生存下来，击败敌人获取经验，升级解锁强大技能
- **操作**：
  - `W/A/S/D` — 移动角色
  - `Space` / `J` — 释放人物技能
  - `Esc` — 暂停
- **特色系统**：
  - 8 种独特武器（自动手枪、回旋镖、轨道剑、闪电链、地雷等）
  - 5 种属性强化（移速、伤害、攻速、生命、拾取范围）
  - 技能协同进化系统
  - 自动攻击，专注走位即可
- **支持平台**：键盘 / 鼠标 / 触摸屏 / 游戏手柄

## 📦 安装方法

### 从 VS Code 扩展市场安装（推荐）

1. 打开 VS Code
2. 按下 `Ctrl+Shift+X` 打开扩展面板
3. 搜索「荆棘收割者」或「Thorn Reaper」
4. 点击「安装」按钮
5. 安装完成后，按 `Ctrl+Alt+T` 即可开始游戏

> 扩展市场安装的版本会自动接收更新，无需手动操作。

### 从 VSIX 文件安装

1. 打开 VS Code
2. 按下 `Ctrl+Shift+X` 打开扩展面板
3. 点击面板右上角的 `···`（更多操作）
4. 选择「从 VSIX 安装...」
5. 选择 `thorn-reaper.vsix` 文件
6. 安装完成后，按 `Ctrl+Alt+T` 即可开始游戏

### 从命令行安装

```bash
code --install-extension thorn-reaper.vsix
```

## 🔧 更新方法

### 市场版本（推荐）

扩展市场安装后会自动检查更新，你也可以手动检查：
1. 打开扩展面板
2. 点击右上角的 `···` →「检查扩展更新」
3. 如有可用更新会自动提示安装

### VSIX 版本

1. 下载最新版本的 `thorn-reaper.vsix`
2. 在 VS Code 扩展面板中，找到「荆棘收割者 — Thorn Reaper」
3. 点击扩展卡片下方的「⚙️ 设置」图标
4. 选择「更新」并选择新的 `.vsix` 文件

或者直接重新安装（覆盖旧版本）：

```bash
code --install-extension thorn-reaper.vsix --force
```

## ❓ 常见问题

**Q: 点击活动栏图标没有反应？**
A: 请尝试重启 VS Code。如果仍不生效，可以使用快捷键 `Ctrl+Alt+T` 或通过命令面板启动。

**Q: 游戏画面卡顿怎么办？**
A: 关闭其他占用 GPU 的程序，或在 VS Code 设置中关闭「Editor: Smooth Scrolling」等视觉效果。

**Q: 为什么游戏是英文的？**
A: 游戏界面为中文，部分调试信息可能为英文，不影响正常游玩。

**Q: 存档在哪里？**
A: 游戏进度自动保存在 VS Code 的全局存储中，卸载扩展不会丢失存档。

**Q: 如何从 VSIX 迁移到市场版本？**
A: 直接安装市场版本会自动覆盖 VSIX 版本，存档不会丢失。
